# 🥕 Fresh Cart – Fruits & Vegetables Web Application
##### 📌 Project Overview

Fresh Cart is a web-based application designed to make buying fresh fruits and vegetables easy and convenient. The platform allows users to browse products, add items to the cart, and place orders in a simple and user-friendly way.

#### 🎯 Objective

The main goal of this project is to create a clean and interactive online shopping experience for daily grocery needs, focusing on fresh produce.

##### 🚀 Features
🛒 Browse fruits and vegetables with images and prices
➕ Add items to cart
➖ Remove or update quantity in cart
💰 View total price dynamically
📱 Responsive design for mobile and desktop
🔍 Simple and easy navigation
##### 🛠️ Technologies Used
Frontend: ReactJS, HTML, CSS, JavaScript
Backend: Node.js (if used)
Database: MongoDB (if used)
Tools: VS Code, Thunder Client
#### ⚙️ How It Works
User visits the homepage
Views available fruits and vegetables
Adds required items to the cart
Cart updates total price automatically
User can review and modify items before checkout

Handling user interactions
Designing responsive UI
Connecting frontend with backend
